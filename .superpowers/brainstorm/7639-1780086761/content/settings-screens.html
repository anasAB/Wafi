Key layout decisions

  • 📱 Mobile: full-screen push navigation — Settings root → Personal sub-screen
  • 🖥 Desktop: sidebar + content panel, no push — clicking "شخصي" in sidebar swaps the right panel inline
  • Segmented controls for theme/text-size (immediate, no Save button needed)
  • Segmented control for language too — change fires immediately, whole app re-renders
  • Sign out is a stub row for now (Epic 5 wires it up)
  • About section is a stub row linking to app version (future Epic 6 story)
