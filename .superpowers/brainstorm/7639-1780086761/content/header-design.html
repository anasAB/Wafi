Code change

AppHeader gets a showSettings prop (default true). When false (e.g. on the Settings page itself), the gear is hidden. The gear is a <RouterLink to="/settings"> — no emit needed.


The existing showBack prop still works. On the Settings page, showBack=true and showSettings=false.
